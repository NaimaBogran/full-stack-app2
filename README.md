**Monologue Library** is a full-stack web platform where actors and performers can **discover, share, and save powerful monologues** for auditions, classes, and creative inspiration.  

The app blends **community-driven discovery** with a clean, creative UI ‚Äî built with Node.js, Express, Passport authentication, and MongoDB. Users can log in, post their favorite monologues, upvote others, and curate their personal library of go-to audition material.

---

## ‚ú® Features

**Community Monologue Board**  
Browse a shared collection of monologues submitted by other actors, with context like genre, tone, and play title.

**Likes & Feedback**  
Upvote your favorite pieces to highlight great writing and performance material.

**User Profiles**  
Create an account, post new monologues, and manage your contributions in a personal dashboard.

**Collaborative Notes** *(coming soon)*  
Allow users to comment, discuss interpretations, and share performance advice.

**Modern Theatrical Aesthetic**  
Built with glassy gradients, soft lighting, and playful color palettes inspired by stage and creativity.

**Modular Full-Stack Structure**  
EJS templating + Express routing + MongoDB collections make this project scalable for new features like search filters, tags, or video embeds.
